## 🌟 项目概述
## 前言

终于把个人博客完整部署上线啦！
折腾了 Astro \+ Firefly 主题、Cloudflare Workers 域名解析、页面美化、侧边栏组件配置一系列流程，踩了不少坑，也收获了一套完全属于自己的轻量化学习站点。

本站地址：`https://xyxstudy.kdns.fr`
搭建时间：2026 年 7 月

## 一、建站初衷

1. **沉淀学习笔记**
日常刷题、Python 开发、算法学习、水利调度建模、深度学习实验会产生大量零散代码与思路，以前存在本地文档很容易丢失，博客可以长期归档，随时查阅。

2. **输出倒逼输入**
看懂代码、看懂理论不等于完全掌握，把知识点整理成文、写下踩坑过程，才能真正吃透内容。

3. **分享干货交流**
记录实用代码案例、环境配置教程、排错解决方案，希望能帮到同样在自学编程、做数据分析的朋友，也欢迎大家留言交流思路。

4. **长期自律记录**
侧边栏的运行时长、文章统计、日历都是给自己的监督，持续更新，稳步提升编程与工程能力。

## 二、站点技术栈说明

### 核心框架与主题

- 静态站点框架：**Astro**

- 博客主题：Firefly v6\.13\.8（轻量化、UI 清爽、自定义自由度高）

- 部署载体：Cloudflare Workers

- 协议许可：CC BY\-NC\-SA 4\.0（非商用可转载，标注来源即可）

### 我自定义修改的内容

1. 首页顶部全屏动漫背景图，柔和渐变遮罩提升阅读舒适度

2. 侧边栏定制：个人简介卡片、公告栏、站点统计、日历组件

3. 导航栏拆分：主页、文章、友链、留言、相册、关于我

4. 精简多余组件，移除冗余播放器，保证页面加载速度

5. 域名绑定、CDN 加速，国内访问更稳定

## 三、后续更新规划

### 内容方向

1. **算法刷题专栏**
LeetCode 滑动窗口、动态规划、KNN 等经典题型，附完整 Python 代码 \+ 思路拆解

2. **Python 工程实战**
数据分析 Pandas/Matplotlib 可视化、能源调度模型、PyQt 桌面程序、PyTorch 深度学习基础

3. **环境搭建踩坑记录**
Conda 虚拟环境、Git/GitHub 部署、Astro 博客配置、Vercel/Cloudflare 静态站排错教程

4. **数学与建模笔记**
数学建模思路、调度优化 DP/DDDP 算法、工程仿真案例整理

### 更新节奏

日常碎片化更新刷题笔记，每周至少一篇完整实战干货，持续填充分类与标签，慢慢丰富站点内容。

## 四、写在最后

这是博客的第一篇文章，也是自学之路的全新起点。
不求流量，只为记录自己一步步成长的痕迹。如果你也同样在自学编程、算法、数据分析，欢迎常来逛逛，留言区随时交流讨论，一起自律深耕，稳步提升技术能力。
