## 一、什么是线性回归
### 1.1 线性回归核心思想：拟合最优直线/超平面
&emsp;&emsp;线性回归是经典监督学习回归算法，核心逻辑一句话概括：寻找一条最优线性超平面，尽可能贴合样本特征与连续标签之间的关系。
模型通过调整系数，最小化预测值和真实标签之间的整体误差；输入特征，输出连续数值。一维特征对应直线，二维特征对应平面，多维特征对应超平面。

### 1.2 线性回归属于监督学习的原因
&emsp;&emsp;线性回归属于典型监督学习，核心依据有三点：一是依赖带连续标注的训练数据，依靠样本真实标签求解最优参数；二是学习特征与标签之间的线性映射关系，显式拟合输入输出函数；三是专注监督学习核心回归任务，用于预测连续数值。

### 1.3 线性回归适用场景
&emsp;&emsp;线性回归模型结构简单、可解释性极强，假设数据存在线性相关关系，适合特征与目标近似线性变化的场景。

一、基础回归预测
&emsp;&emsp;房价预测、销量预估、营收预测、气温拟合、能耗预测等。优势：模型轻量化、训练速度极快、参数具备业务解释性；劣势：只能捕捉线性关系，无法拟合非线性规律。

二、基准基线模型
&emsp;&emsp;搭建机器学习项目时，常先用线性回归作为基线，后续对比树模型、神经网络等复杂模型的提升效果。

三、特征分析
&emsp;&emsp;系数正负代表特征影响方向，系数大小代表影响强弱，用于业务因素分析。


## 二、线性回归核心数学表达
### 2.1 模型表达式
一元线性回归（单个特征）：
$$\hat{y} = w_0 + w_1 x_1$$

多元线性回归（多个特征）：
$$\hat{y} = w_0 + w_1 x_1 + w_2 x_2 + \dots + w_n x_n$$
向量形式（工程常用）：
$$\hat{y} = \boldsymbol{w}^T \boldsymbol{X}$$
- $\hat{y}$：模型预测值
- $w_0$：截距（偏置项 bias）
- $w_1,w_2\dots w_n$：特征权重系数

> 重点：**线性指参数线性，不是特征线性**；特征可以构造多项式得到多项式回归。

### 2.2 损失函数：最小二乘法（平方误差损失）
衡量预测值与真实值整体偏差，线性回归标准损失：均方误差 MSE
$$
\mathcal{L} = \frac{1}{m}\sum_{i=1}^m (y_i - \hat{y}_i)^2
$$
目标：寻找一组参数 $\boldsymbol{w}$，使损失函数取值最小。

求解两条主流方案：
1. **解析解（正规方程）**：直接数学公式求出最优参数，无需迭代；特征不多时高效。
2. **梯度下降**：迭代逐步更新权重，大数据、高维特征场景首选，也是深度学习基础优化方式。

### 2.3 模型基础假设（经典高斯马尔可夫假设）
1. 特征与目标存在近似线性关系；
2. 误差均值为0；
3. 误差方差齐性（同方差）；
4. 误差之间相互独立；
5. 误差服从正态分布。
假设不满足时模型效果会明显下降。

### 2.4 衍生正则化线性模型（解决过拟合）
原始最小二乘无约束，特征过多、多重共线性时容易过拟合，衍生三类常用变体：
1. **普通线性回归 LinearRegression**：无正则
2. **岭回归 Ridge**：L2正则，约束权重不要过大，保留全部特征
$$
\mathcal{L}_{ridge} = \frac{1}{m}\sum_{i=1}^m (y_i - \hat{y}_i)^2 + \alpha\sum_{j=1}^n w_j^2
$$
3. **Lasso回归**：L1正则，可以产生稀疏权重，自动实现特征筛选
$$
\mathcal{L}_{lasso} = \frac{1}{m}\sum_{i=1}^m (y_i - \hat{y}_i)^2 + \alpha\sum_{j=1}^n |w_j|
$$
4. **弹性网 ElasticNet**：L1+L2结合，融合岭回归与Lasso优势

### 2.5 三类正则回归适用场景对比
1. **LinearRegression**
特征数量少、无多重共线性、样本充足；只适合简单基线建模。
> 缺点：无法抑制过拟合，多重共线性下参数不稳定。

2. **Ridge 岭回归（L2正则）**
特征高度相关、存在多重共线性，希望保留所有特征；工业使用最多。
特点：权重被压缩，但不会等于0。

3. **Lasso（L1正则）**
特征维度很高，希望模型自动剔除无关特征，实现特征选择。
特点：可以将不重要特征系数压缩至0；高度相关特征容易随机挑选其一。

4. **ElasticNet 弹性网**
高维特征、特征存在相关性，兼顾特征筛选与稳定训练。

## 三、关键超参数与训练要点
### 3.1 核心超参数 α（正则系数）
$\alpha$ 控制正则惩罚强度：
- $\alpha=0$：等价原始最小二乘；
- $\alpha$ 越大：惩罚越强，权重收缩越明显，降低复杂度，缓解过拟合；
- $\alpha$ 过小：正则几乎失效，依旧容易过拟合。

### 3.2 特征标准化必要性
&emsp;&emsp;线性回归（尤其带正则的 Ridge/Lasso/ElasticNet）**强烈建议特征标准化 StandardScaler**。
原因：正则项同等惩罚所有权重；原始特征量纲不一致时，数值尺度更大的特征会天然受到更强惩罚，造成训练偏差。

> 普通LinearRegression解析解不受量纲影响；但只要使用正则、梯度下降优化，必须标准化。

### 3.3 多重共线性问题
多个特征之间高度相关（多重共线性）：
- 普通线性回归：参数震荡、解释性变差；
- 解决方案：特征筛选、降维PCA、优先使用岭回归Ridge。

### 3.4 模型评估指标（回归任务专用）
线性回归常用评价指标，用于挑选最优超参数：
1. MSE 均方误差
$$MSE=\frac{1}{m}\sum(y_i-\hat{y}_i)^2$$
2. RMSE 均方根误差（和标签量纲一致）
$$RMSE=\sqrt{MSE}$$
3. MAE 平均绝对误差，对异常值更加鲁棒
$$MAE=\frac{1}{m}\sum|y_i-\hat{y}_i|$$
4. $R^2$ 决定系数，值域 $(-\infty,1]$，越接近1拟合效果越好

### 3.5 最优超参数选取方式
采用**K折交叉验证 + 网格搜索GridSearchCV/随机搜索RandomizedSearchCV**，遍历不同α取值，选择验证集 $R^2$ 最高、误差最低的参数。

## 四、sklearn 代码调用
> **注意**：Ridge、Lasso、ElasticNet 使用前建议标准化特征；普通LinearRegression解析解不强制标准化，但工程实践统一标准化养成好习惯。
- 基础线性回归 LinearRegression
```Python
# 导入
from sklearn.linear_model import LinearRegression
# 初始化模型
lr = LinearRegression()
# 训练
lr.fit(X_train, y_train)
# 预测
y_pred = lr.predict(X_test)
# 获取参数
print("权重系数：", lr.coef_)
print("截距：", lr.intercept_)
```

- 岭回归 Ridge（L2正则）
```Python
from sklearn.linear_model import Ridge
ridge = Ridge(alpha=1.0)
ridge.fit(X_train, y_train)
y_pred = ridge.predict(X_test)
```

- Lasso回归（L1正则，特征筛选）
```Python
from sklearn.linear_model import Lasso
lasso = Lasso(alpha=0.1)
lasso.fit(X_train, y_train)
y_pred = lasso.predict(X_test)
```

- 弹性网 ElasticNet
```Python
from sklearn.linear_model import ElasticNet
# l1_ratio控制L1占比，0等价Ridge，1等价Lasso
enet = ElasticNet(alpha=0.1, l1_ratio=0.5)
enet.fit(X_train, y_train)
y_pred = enet.predict(X_test)
```

# 五、线性回归优缺点总结
## 5.1 算法优势
1. 原理清晰，数学理论完善，极易理解；
2. 训练速度极快，预测开销小，支持大规模数据；
3. 模型具备强可解释性：权重系数直观反映特征影响方向与强弱；
4. 存在解析解，训练结果稳定；可作为复杂模型的强基线；
5. 拓展灵活：增加正则项、构造多项式特征即可升级。

## 5.2 算法局限性
1. 假设特征与目标为线性关系，难以捕捉非线性、复杂交互规律；
2. 对异常值敏感，异常样本会大幅偏移拟合直线；
3. 无法自动学习特征交互，需要人工构造交叉特征；
4. 存在多重共线性时，基础线性回归参数不稳定；
5. 不擅长高维稀疏原始特征场景（需配合正则、特征工程）。
